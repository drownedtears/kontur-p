Инструкция по использованию программы "Сервер Контур-П2" 
*****************************************************************************
НАСТРОЙКА ПРОГРАММЫ:
1. Программа должна быть установлена на диск, где находится "Контур-П2";
2. Для надёжности "Общий доступ должен быть закрыт";
3. При первом запуске программы необходимо выбрать корневую папку, где находится "Контур-П2";
4. При изменении местоположения "Контур-П2", необходимо в программе "Сервер Контур-П2" зайти во вкладку "Настройка" -> "Выбор пути Конткр П"
	4.1 Выбрать новый путь "Контур-П2";
	4.2 После выбора пути, нажать повторно "Настройка" -> "Выбор пути Конткр П";
*****************************************************************************
ИСПОЛЬЗОВАНИЕ ПРОГРАММЫ:
5. При двойном нажатии на пользователся в "списке студентов в сети", появится информация о студенте в отдельном окне;
6. Для отправления данных, необходимо набрать данные в окне "Данные для КП2";
7. Нажать кнопку "Отправить данные пользователям";
*****************************************************************************
ОСОБЕННОСТИ ПРОГРАММЫ:
8. При нажатии "Файл" -> "Все пользователи" откроется окно, которое содержит информацию о всех пользователях, использующих программу "Контур-П2";
9. Для вывода информации о пользователе, необходима двойным щелчком нажать на пользователя в списке "Список всех пользователей";
10. При нажатии на кнопку "Обновить", добавятся пользователи, зашедшие после запуска "Сервер Контур П2";
11. Для удаления нужного пользователя необходимо поставить галочку на против ее фамилии или выделить всех, нажав на кнопу "Выделить всех";
*****************************************************************************
